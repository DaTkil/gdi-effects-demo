# GDI Effects Demo (Windows Visual Effects)

This project is a harmless visual effects demo using GDI-based screen effects on Windows.

It is for educational and experimental purposes only.

## What it does
- Uses Windows GDI-based drawing effects
- Generates visual distortions and overlays
- Runs locally on the user's machine only
- Does NOT access files, network, or system settings

## How to run
Double-click `run_chaos.bat` or run:
python chaos.py

## How to stop
- Press Ctrl + C in the terminal
- Or close the command window

## Status
This project is currently in beta and may be updated in the future.
